You can start from [Workers](https://developers.cloudflare.com/workers/get-started/guide/) or clone an Example.

## Install

```shell
npm i discord-hono
npm i -D discord-api-types # When using TypeScript
npm i -D dotenv # When using 'npm run register'
```

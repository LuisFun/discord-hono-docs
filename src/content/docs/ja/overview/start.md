[Workers](https://developers.cloudflare.com/workers/get-started/guide/) から始めてもいいし、Example をクローンしてもいい。

## Install

```shell
npm i discord-hono
npm i -D discord-api-types # TypeScript を使うとき
npm i -D dotenv # 'npm run register' を使うとき
```
